feat: Complete Stripe Connect payment processing integration

This commit implements a full Stripe Connect payment processing system with
code quality improvements and performance optimizations.

## Stripe Integration Features
- Stripe Connect Express accounts for multi-tenant payment processing
- Payment intent creation API with amount validation and balance checking
- Secure payment form component using Stripe Elements
- Webhook handler for real-time payment status updates
- Automatic invoice status updates on successful payment
- Support for partial payments with validation
- Payment confirmation email automation
- Stripe Connect onboarding flow in settings page
- Platform fee calculation and application
- Payment error handling and user feedback

## Code Quality Improvements
- Updated Stripe API version to latest (2025-12-15.clover)
- Added proper cleanup and abort controllers in payment form
- Implemented payment amount validation to prevent overpayment
- Optimized settings page polling (conditional polling instead of constant)
- Improved TypeScript types and removed 'any' types where possible
- Enhanced error handling with proper type checking
- Fixed idempotency checks in webhook handler
- Added proper invoice relation handling in webhooks

## Performance Optimizations
